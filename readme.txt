Easy to use WooCommerce price display anywhere 

== Description == 

This is a cool plugin to display just product price in any WooCommerce based website. It is lightweight and easy to use. 

Please note: 
- This is a WooCommerce based plugin, so WooCommerce plugin must be activated before active this plugin. 
- You must create a product to display the price of that product. 

= Using the Plugin = 

* Download the plugin, install and active.

* Now Add new product and publish the product. finds its product id, for exmaple :  https://domain.com/wp-admin/post.php?post=2861&action=edit  Here 2861 is your Product ID

* Now you are ready to use this shortcode. Just type this shortcode by replacing the ID and make sure you use ID of right product [zit_display_price id="2861"]

* That's it. You are ready to go!

== Installation ==

Installation process is very easy. Follow the steps bellow:

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly by search <Strong>WooCommerce price display anywhere</strong>.
2. Then activate the plugin.
3. Just type this shortcode by replacing the ID and make sure you use ID of right product [zit_display_price id="2861"]


== Frequently Asked Questions ==

= what is :WooCommerce price display anywhere"? =
Sometimes you need to display just product price with your customer design so that you can show the price at multicurrency. Our plugin will help you to display WooCommerce Product price display anywhere at your WordPress theme. if you use multicurrency plugin it will help you to display at multicurrency.

= Is it WooCommerce dependent plugins? =
Yes, You must install and active WooCommerce plugins before active this plugin.

= Does it support multicurrency? =
yes you can use this plugin with any multicurrency plugin. thats the best part of this plugin.

= Is it secured? =
100%, Maintained high level validation in both front-end and back-end.
